# Selenium Automation Framework

This is a robust UI automation testing framework built using **Selenium WebDriver**, **Java**, and **TestNG**. It is integrated with a CI/CD pipeline via GitHub Actions to run regressions automatically on code changes and host interactive test execution dashboards.


---

Tech Stack & Tools

* **Language:** Java 17
* **Browser Automation:** Selenium WebDriver 4
* **Test Engine:** TestNG
* **Build Management:** Maven
* **Reporting:** Extent Reports

---

Getting Started Locally

Prerequisites
Ensure you have the following installed on your local machine:
* Java Development Kit (JDK 11 or 17)
* Apache Maven

Execution Steps
1. Clone this repository to your local system.
2. Open your terminal inside the project root folder.
3. Run the following command to execute all test cases:
   ```bash
   mvn clean test
   ```
